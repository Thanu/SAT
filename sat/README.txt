1)Build project with external libraries provided in "lib" folder.
2)Development Tool : Eclipse,Netbeans (Netbeans preferable)
3)Please install WordNet
4)In the property file, put the WordNet installed path upto dict folder in the variable path "wordNetDbDirectory".
Ex : E:/Programs/WordNet/dict
5)select image path from the project folder & put the path into the variable path "imagePath".
Ex : "E:/SAT/sat/img" (the images are in the project folder)
